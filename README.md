# Parking-Payment-System
## Overview
This project is a web-based Parking Payment System that allows users to:

- Register and log in
- Park a vehicle
- Calculate parking fees
- View parking history

## Features

- User authentication (login & register)
- Secure session management
- Parking fee calculation
- Parking history tracking
- Admin/user authorization

## System Architecture

The application follows a simple MVC-style architecture.

Client
- HTML
- CSS
- JavaScript

Server
- Node.js
- Express.js

Database
- MongoDB

Flow
User → Browser → Express Server → Database

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express

Database
- MongoDB

## Installation

Clone the repository

git clone https://github.com/shangc1997/Parking-Payment-System.git

cd Parking-Payment-System

npm install

npm start

Open in browser

http://localhost:3000
